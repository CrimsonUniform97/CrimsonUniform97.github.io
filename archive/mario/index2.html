Infinite Mario Bros!



To play the game, click the window when it's loaded, and press 's'.
You need Java 5.0 to play Infinite Mario Bros.
Click here for an unscaled version

Controls
Use the arrow keys to move mario around.
Press A to run faster and to pick up shells.
Press S to jump.
Press F1 to toggle Scale2x scaling.

About Infinite Mario Bros
Infinite Mario Bros was made by Markus Persson for a Super Mario themed java programming contest.
