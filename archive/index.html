Archive

This is the main directory for everything that we have archived. While we primarily archive the works of Markus "Notch" Persson, we do archive other projects, like Infiniminer. Below, you'll find all the projects in our archive, sorted by release dates in chronological order. If you're looking for something, use the search feature (Ctrl+F).

The early ages (1987-199X/200X)

Childhood

According to an interview, Markus Persson started coding at the age of 7 - that was around 1986 or 1987. Notch's first game was a "very basic text adventure" he made at the age of eight, 1987 or 1988. It is currently unknown if any of these games or notch's first Commodore 128 are still with Notch or his parents.

BitSeven Productions Era

Notch used to make games under his studio's name "BitSeven Productions". He shared screenshots and some basic info about the games in an imgur post, but for your comfort, you can check them out by going here. None of these games were probably publicly released.

The Intra Clinic (2001-2002)

2001

Around 2001, Notch bought the Intra Clinic website (theintraclinic.com) and hosted his work there. This includes games, music, and of course - cryptic messages.

  • Legend of the Chambered was the first known game released on theintraclinic in July 2001, and is also the first known game with this title, as more were to come. Multiple versions of it existed, as written in the changelog. If you want to read more about this, visit the original page, located here. This game is currently lost.

  • SiteApplet was an applet that would load if you entered the website. It was an animated logo of The Intra Clinic. We have found this on archive.org, but can't figure out how to run it for now. If we ever do, we will make sure to update this.

  • Droj was a game hosted on Notch's computer, which suggests it was multiplayer. Nothing is known about Droj, as it is currently lost and the archived site doesn't contain much information.

    • 2002

  • DachonB was a multiplayer gravity thruster type game with collision and sound engine. It was released around May. It is currently lost, including the site and the JNLP file.
    Here's the original description of it by Notch:
    "It’s pretty much a normal thruster or gravity type game, except no singleplayer at all. Some of the cooler thing in there is the collision detecion, the beanshell-enabled console and the sound engine. I was doing this at the same time I was doing my j3d doom engine, and it was about then I realized j3d was way too slow to make anything “real” in. In my humble opinion, of course, so you j3d zealots can drop the pitchforks."

  • Stark was a DOOM clone made in Java released around May. It could load WAD files. The JAR file, the J3D version and the source code are lost, but the rest of the website is archived. Notch abandoned this project after not being able to fix a sprite issue.

  • Untitled Re-coloring Project.

    • Mojang AB (2003-200X)

    2003

  • WURM Online is a MMO RPG that Notch worked on with a few other people, who together founded the original Mojang AB. There is a lot of stuff about this game, so you should check out the dedicated page we have for it.

  • Mojang SiteApplet was an applet that would load if you entered the Mojang website. It was an animated logo of Mojang.

    • 2004

    There are no known projects other than Wurm that notch worked on during this year.

    2005

  • SonicRacer4k was the first time notch attempted to join the 4k Java Competition in January 2005. It was a Sonic game with stolen sprites. Some of the code was based on Stark. You can read the original thread here. It later mutated into a voxel game engine, as said here. Notch forgot to make a backup of the racer source code, so he abandoned it.The game is currently lost, with one screenshot existing.

  • Dungeon4k a game for the j4k competition. If you wish to run it, it is recommended you download this fixed jar, and simply double click it.
