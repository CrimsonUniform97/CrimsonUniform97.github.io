This is the main directory for everything that we have archived. While we primarily archive the works of Markus "Notch" Persson, we do archive others, like Infiniminer. See the topic directory below for all the games in that topic, or take a look at the list of featured games.

The early ages (1987-199X/200X)

Childhood

According to an interview, Markus Persson started coding at the age of 7 - that was around 1986 or 1987. Notch's first game was a "very basic text adventure" he made at the age of eight, 1987 or 1988. It is currently unknown if any of these games or notch's first Commodore 128 are still with Notch or his parents.

DOS Era

Notch used to make games under his studio's name "BitSeven Productions". He shared screenshots and some basic info about the games in an imgur post, but for your comfort, you can check them out by going here. None of these games were probably publicly released.

The Intra Clinic (2001-2003)

2001 - The Intra Clinic's rise

Around 2001, Notch bought the Intra Clinic website (theintraclinic.com) and hosted his work there. This includes games, music, and of course - cryptic messages.

  • The first known game here released here was Legend of the Chambered. Notch made multiple games with this name, and this is the very first one. It is sadly currently lost, but you can check out the original site, or a later version of it. The game had multiple versions, most likely starting with 0.1.0, but the earliest with proof is 0.1.4 thanks to the changelog.
  • Notch also made an applet that would load if you entered the website. It was an animated logo of The Intra Clinic. We have found this on archive.org, but can't figure out how to run it for now. If we ever do, we will make sure to update this.
  • Notch then made a multiplayer RPG-like game called Droj. It is currently lost and the archived site doesn't contain much information.
