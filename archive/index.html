Archive

This is the main directory for everything that we have archived. While we primarily archive the works of Markus "Notch" Persson, we do archive others, like Infiniminer. See the topic directory below for all the games in that topic, or take a look at the list of featured games.

Name Type Released Status Source Description
BitSeven Games DOS Games Unknown Lost (Probably unreleased) Imgur (Archived) Games that Notch made under his old "company" BitSeven.
Legend of the Chambered (2001) Java Game July 2001 Lost (The website is archived, however the game and all the versions are lost.) The original intraclinic website The first documented Legend of the Chambered game. It had mutiple versions, but only 3 are confirmed to exist (0.1.4, 0.1.5 and 0.2.0) thanks to a changelog. Version 0.2.1 was in development, but we don't have any proof of it being released.
Intra Clinic Site Applet Java App Around 2001 Potentionally found (Still figuring it out.) The original intraclinic website An applet that started when you loaded the website.
Droj Java Game 2001 Lost (Due to it being hosted on Notch's PC, however we have the website archived.) The original intraclinic website Not much is known about Droj, all we really know is that the servers for it were being hosted on Notch's computer.
DachonB Java Game Around May 2002 Lost (Including the webpage, the JNLP and screenshots) jvm-gaming It is a j3d version of a game Notch made for work.
"It’s pretty much a normal thruster or gravity type game, except no singleplayer at all. Some of the cooler thing in there is the collision detecion, the beanshell-enabled console and the sound engine. I was doing this at the same time I was doing my j3d doom engine, and it was about then I realized j3d was way too slow to make anything “real” in. In my humble opinion, of course, so you j3d zealots can drop the pitchforks."
Stark Java Game Around May 2002 Lost (The website and a JNLP file are archived. The game, source code and the j3d version are lost.) jvm-gaming Stark is a java 2.5d graphics engine which loads DOOM WADs.
Wurm Online Java Game 2003 TBA TBA Wurm Online is a MMO RPG game that Notch used to code, when he first founded Mojang Specifications with Rolf. More info coming soon.
Mojang Site Applet Java App December 2003 Potentionally found (Still figuring it out.) jvm-gaming An applet that started when you loaded the website. Had multiple versions.
SonicRacer4k Java Game January 2005 Lost (Only one screenshot is archived.) jvm-gaming A sonic game with stolen sprites. It later evolved into a voxel cave engine. Notch forgot to save the source code for the racer.
Dungeon4k Java Game January 2005 Archived (Including the website, screenshots, JNLP and JAR.) jvm-gaming Navigate through a dungeon and collect as many diamonds as possible.
Hunters4k Java Game January 2005 Archived (Including the website, screenshots, and JAR.) jvm-gaming An action packed first person shooter.
