AntsInThePants (http://www.angelfire.com/retro/gdog/ants/ants.html/)
By carmichaelbaby
Applet
This game was submitted in the Java4K 2003 Competition.

Description

Try to swat the ants up or down with the swatter to make them go in the pants
Red ants are worth 3, black ants 1, and blue ants 0 points
If you hit a blue ant, the swatter will be stuck. Wiggle the mouse around to unstick it.
After 6 groups of ants has gone by, bug poison will cause the ants that escaped to go down to zero
If 25% of the ants escape before the 6th group has gone by, the game is over.

Instructions

Use mouse to move the swatter.

Installing

Either open the page with a browser that supports applets, or download this page and the JAR (make sure they are in the same folder!) and run them using appletviewer: appletviewer index_original.html.
