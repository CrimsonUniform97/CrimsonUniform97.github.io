Droj

If you see a small Droj-logo here (), the Droj servers are up and running, and you can start playing by clicking here.
Most of the time, however, the servers will be down. This is because I host Droj from my home PC.

When Droj asks you to log in, just enter your desired username and a random password. Currently, all logins are accepted as there is no database anywhere to be found on my computer.
Note that you will need Java 2, Standard Edition version 1.3 or higher to play Droj.

If you find any bugs or have any questions, please contact me.
